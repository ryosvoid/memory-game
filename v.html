Coordonnées, matrices associées et base

Coordonnées dans la base B et matrice associée :
Les coordonnées des vecteurs dans la base B sont : 
[v₁]B = (2, -1, 3)
[v₂]B = (0, 4, -2)
[v₃]B = (1, 1, 1)
La matrice associée à S = (v₁, v₂, v₃) est : 
AS = | 2  0  1 |
               | -1 4  1 |
               | 3 -2  1 |
Rang de la matrice et justification :
Le rang de AS vaut 3, donc les 3 vecteurs sont linéairement indépendants.
Justification : On voit par exemple que le déterminant de la matrice 3x3 est non nul, ou on le réduit par Gauss à une forme échelonnée, ou on constate que chaque vecteur apporte une direction nouvelle.
Conclusion : S = (v₁, v₂, v₃) est une base de ℝ³.

Base du sous-espace engendré par F

Les vecteurs v₁ et v₃ sont non colinéaires (car aucun n’est un multiple de l’autre), ils forment donc une base du plan qu’ils engendrent.
Dès que deux vecteurs ne sont pas colinéaires, ils sont indépendants.
Conclusion : (v₁, v₃) est une base du sous-espace engendré par F.

Vérification par la méthode de la forme échelonnée (réduction par colonnes)

On forme la matrice A ∈ M3,4(ℝ) dont les colonnes sont v₁, v₂, v₃, v₄ : 
A = | 1 2 0 1 |
    | 2 4 1 3 |
    | 0 1 1 4 |
On applique la méthode de Gauss (réduction par colonnes) pour vérifier la liberté des colonnes :
On voit rapidement que v₂ = 2v₁, donc la colonne 2 est combinaison des autres.
Les autres colonnes (v₁, v₃, v₄) sont-elles indépendantes ?
On remarque que v₄ = v₁ + v₃.
Donc, l’espace engendré est de dimension 2, base : (v₁, v₃).

Exercice 3

Vérification par observation de combinaisons linéaires :
On a : 
v₁ = (1,2,3)
v₂ = (2,4,6)
v₃ = (0,1,1)
v₄ = (1,3,4)
On remarque :
  • v₂ = 2 v₁
  • v₄ = v₁ + v₃
Donc v₂ et v₄ sont des combinaisons linéaires de v₁ et v₃.
Conclusion : (v₁, v₃) est une base de l’espace engendré par F.

Exercice 4

Famille F = {v₁, v₂} avec v₁ = (1,0,1), v₂ = (0,1,1)
Montrons que F est libre :
Supposons λ v₁ + μ v₂ = (0,0,0), soit : 
λ(1,0,1) + μ(0,1,1) = (0,0,0)
  • Première coordonnée : λ = 0
  • Deuxième : μ = 0
  • Troisième : λ + μ = 0 ⇒ 0 + 0 = 0
Seule solution : λ = μ = 0.
Conclusion : (v₁, v₂) est libre.
